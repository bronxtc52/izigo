# ТЗ: Knowledge Base + AI-ассистент для партнёров IziGo

**Дата:** 2026-06-24  
**Статус:** черновик → на ревью Гейт 1  
**Фича-флаг:** `ai_assistant` (deny-by-default)

---

## Цель

Партнёр задаёт вопрос на русском или английском прямо в Telegram Mini App и получает чёткий ответ из базы знаний сервиса — без необходимости открывать тикет в поддержку или гуглить.

Инструмент снижает нагрузку на helpdesk по типовым вопросам: «как считается бонус», «как пополнить», «что такое Gold» и т.п.

---

## Пользователи и сценарии

**Аудитория:** партнёры (Members) в Telegram Mini App.

| Сценарий | Вопрос партнёра | Ожидаемый ответ |
|---|---|---|
| S1 | «Как считается бинарный бонус?» | Объяснение 5% от малой ветки + пример |
| S2 | «Что нужно для ранга Manager Silver?» | Требования: 8000 PV малой ветки, 3 менеджера в команде |

| S5 | «Чем Gold лучше Bronze?» | Сравнение пакетов по цене и рефбонусу |
| S6 | «Что такое PV?» | Определение Partner Volume |
| S7 | «Реферальная ссылка — где взять?» | Профиль → Реф-ссылка |

## Что строим

### A. Knowledge Base (KB)

Четыре MD-файла в `mh-calc-backend-main/resources/knowledge-base/`:

| Файл | Содержание |
|---|---|
| `marketing-plan.md` | Пакеты, ранги, бонусы (binary/referral/leader/rank) с реальными числами |
| `faq.md` | Регистрация, оплата, вывод, KYC, реф-ссылка, технические вопросы |
| `onboarding.md` | Первые шаги нового партнёра: купить → подключить кошелёк → пригласить |
| `technical.md` | TON Pay, статусы платежа, подключение кошелька, Mini App |

Файлы — версионируемые, редактируемые без деплоя (при условии копирования в образ).

### B. Backend-эндпоинт

`POST /api/cabinet/assistant/ask` (middleware: `telegram.auth`)

**Запрос:**
```json
{"question": "Как считается бинарный бонус?", "locale": "ru"}
```

**Ответ:**
```json
{"status": "success", "answer": "Бинарный бонус составляет 5%..."}
```

**Логика:**
1. Валидация: `question` ≤ 500 символов, `locale` ∈ {ru, en}.
2. Загрузить 4 KB-файла из `resources/knowledge-base/`.
3. Собрать системный промпт (роль + KB + контекст пользователя: ранг, пакет, баланс).
4. Вызвать Claude API (`claude-haiku-4-5`) с вопросом.
5. Ответ вернуть как plain text (без markdown — sanitized для Telegram HTML).

**API-ключ:** KV-секрет `izigo--prod--ANTHROPIC-API-KEY`.  
Rate limit: 10 запросов/минуту на партнёра (429 при превышении).

### C. Frontend

Новая вкладка «Помощник» в Mini App (за feature flag `ai_assistant`):

- Список подсказок (suggested questions) при пустом чате.
- Поле ввода + кнопка «Спросить».
- История сообщений текущей сессии (в памяти браузера, без персиста).
- Индикатор загрузки (Spin).
- Стиль: Aurora palette из `tokens.js`.

## Ограничения и границы

- **Движок расчёта (Modules/Calculator) не трогаем** — KB описывает правила словами.
- **Нет векторного поиска** — KB помещается в контекст Haiku (~8k токенов) целиком.
- **Нет памяти сессии на бэке** — каждый запрос независим (новый messages-массив).
- **Нет редактора KB в UI** — только редактирование MD-файлов + деплой.
- **Нет истории чатов** в БД — не логируем вопросы/ответы (v1).
- **Только Telegram Mini App** — не в веб-админке.
